# D&D NLP Assistant

This project explores how natural language processing techniques can simplify complex tabletop RPG rules for beginners.

Many new players struggle to understand dense rulebooks. This assistant retrieves rule descriptions and generates simplified explanations to help users understand mechanics quickly.

## Example

User question:

"How does an attack roll work?"

Assistant output:

"When a character attempts to hit a target, they roll a 20-sided die and add bonuses..."

## Goal

Lower the barrier to entry for new players learning tabletop RPG systems by translating rule text into beginner-friendly explanations.

## Future Improvements

- Use a vector database for rule retrieval
- Implement transformer-based summarization
- Expand rule dataset
